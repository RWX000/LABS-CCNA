# LAB 2 CCNA

## Objective

### Divide a network provided by an ISP into 8 smaller subnets
-Basic configuration on devices  
-Subnetting (FLSM)  
-Static routing  
-Remote access (SSH)   
-Trunk port configuration  
-Router-on-a-Stic

## Materials / Topology
- The topology for each network will be star type  
- Router 1 (Network one)
- Router 1 (Network two) 
- SW 3 (Network one)
- SW 1 (Network two)  
- PCs  32 
- Serial cables 1 
- HTTP servers 2
