# LAB 2 CCNA

## Objetivo
### Dividir una red provista por un provedor (ISP) en 8 subredes mas pequeñas

-Subnetting (FLSM)
-Enrutamiento estatico
-Acceso remoto (SSH)
-Vlans (Soporte,Administracion,Ventas)
-Configuracion del puerto Trunk
